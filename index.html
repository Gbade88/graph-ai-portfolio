About the Course

This 8-week course explores practical applications of generative AI. It focuses on using no-code AI tools to automate tasks, analyze data, and enhance productivity in professional and research settings.

M1: Websites, Docs & Presentations

Learn how LLMs work, generate text, create no-code websites, and design presentations with AI.

M2: AI-Automated Data Analysis

Master AI tools for data analysis, file manipulation, and creating automated reports.

M3: GenAI for Research & Text Analysis

Explore AI for literature search, text analysis, and knowledge synthesis to accelerate information gathering.

M4: Custom GPTs and Chatbots

Build custom chatbots for research assistance and learn about advanced LLM architecture.

M5: AI for Task Automation

Discover strategies for automating repetitive tasks, setting up email automations, and building AI agents.

M6: GenAI for Communication

Learn to use AI for research dissemination, creating visuals, and producing explanatory content.
